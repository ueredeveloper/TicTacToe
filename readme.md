# Tic Tac Toe - Jogo Da Velha



## Objetivo

  Simular o jogo no Windows através do cmd. Utilizando

o prompt comando e o teclado numérico da seguinte forma:

 

 X O X   |   7 8 9 <br>

 O X O   |   4 5 6 <br>

 X O X   |   1 2 3 <br>
